Topix.filter_active — Method
filter_active(topixdf) -> DataFrame
filter_active!(topixdf) -> DataFrame

最新の日付の銘柄のみを取得する。

Example

topixdf = CSV.File("topixweight_j.csv") |> DataFrame
df = filter_active(topixdf)
allequal(df.日付)

# output
true
